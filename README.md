## ✨ О проекте

Данный русификатор обеспечивает **полный перевод** всех элементов интерфейса приложения [MyDockFinder](https://www.mydockfinder.com/) — macOS-подобной панели Dock и строки Finder для Windows.

Переведено всё — от системных уведомлений и меню настроек до виджетов погоды, управления питанием и контекстных меню.

---

## 📦 Что включено в перевод

| Раздел | Описание |
|---|---|
| 🖥️ **Интерфейс Дока** | Иконки, метки, режимы отображения |
| ⚙️ **Настройки** | Все параметры и опции программы |
| 🔔 **Уведомления** | Системные сообщения и предупреждения |
| 📅 **Календарь** | Шрифт, цвета, положение |
| 🌦️ **Погода** | Все погодные условия |
| 🔋 **Батарея и сеть** | Статусы зарядки, Wi-Fi |
| 🚀 **Launchpad** | Управление иконками |
| 🖱️ **Контекстное меню** | Все пункты меню |
| 💤 **Питание** | Сон, гибернация, выключение, выход |
| 🌐 **Finder** | Строка меню, вкладки, подсказки |

## 🚀 Установка

> **Требования:** MyDockFinder любой актуальной версии

### Шаг 1 — Скачайте файл

Скачайте файл **`Russian.ini`** из раздела [Releases](../../releases) или прямо из репозитория.

### Шаг 2 — Поместите файл в папку `lang`

Скопируйте скачанный файл `Russian.ini` в следующую директорию:

```
📁 MyDockFinder\          ← корневая папка приложения
    └── 📁 lang\          ← сюда помещаем файл
            └── 📄 Russian.ini
```

> 💡 **Путь по умолчанию** обычно выглядит так:
> ```
> C:\Program Files\MyDockFinder\lang\Russian.ini
> ```

### Шаг 3 — Выберите язык в настройках

1. Откройте **Настройки MyDockFinder** (правый клик по Доку → *Настройки*)
2. Перейдите во вкладку **Основные**
3. В пункте **«Язык интерфейса»** выберите **Russian**
4. Перезапустите программу
